# 🛒 Grocery Billing System

## 📌 Features
- Add products to bill
- Auto price fetch from database
- Total calculation
- Generate bill
- QR Code payment support
- Print bill

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: MySQL

## ▶️ How to Run
1. Import database.sql in MySQL
2. Start XAMPP / WAMP
3. Place project in htdocs
4. Open frontend/billing.html

## 💳 Payment
QR Code available after bill generation
